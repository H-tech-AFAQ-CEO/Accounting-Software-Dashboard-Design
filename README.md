## Features

### Core Dashboard Components

- **Vertical Sidebar Navigation** — Fixed left navigation with collapsible menu on mobile.
- **Centered Main Content** — Clean, focused layout with max-width container.
- **Responsive Design** — Works seamlessly on desktop, tablet, and mobile.
- **Dark Theme** — Modern navy/dark color scheme optimized for finance dashboards.

### Key Sections

| Section | Description |
|---------|-------------|
| **Stats Cards** | Awaiting payment, overdue bills, revenue MTD, active projects |
| **Bills to Pay** | Bar chart visualization + sortable bills table with status badges |
| **Upcoming Deadlines** | Tax & compliance calendar with status indicators |
| **Expense Breakdown** | Progress bars showing category spending |
| **Quick Actions** | 6 action cards (Create Invoice, Log Expense, Sync Bank, etc.) |
| **Recent Activity** | Real-time feed of invoice payments, bills, and expenses |
| **Top Clients** | Revenue breakdown by client |

### Interactive Elements

- Active navigation state highlighting
- Hover effects on cards, buttons, and table rows
- Mobile hamburger menu with overlay
- Clickable "Pay", "Review", and "View" action links
- Responsive bar chart (pure CSS)
